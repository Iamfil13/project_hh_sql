# Анализ рынка вакансий Data Science (SQL + Python)

## Описание проекта

Данный проект посвящён анализу рынка вакансий на основе базы данных с вакансиями и работодателями (данные, аналогичные HH.ru).  
Основной фокус исследования — **вакансии в сфере Data Science**, требования к кандидатам, уровень зарплат, география рынка и общие тенденции рынка труда.

Проект выполнен в формате аналитического исследования с использованием **SQL** для извлечения данных и **Python (pandas)** для работы с результатами запросов и интерпретации.

---

## Цели проекта

- Проанализировать структуру рынка вакансий в целом  
- Выделить и изучить вакансии, относящиеся к Data Science  
- Ответить на ключевые вопросы:
  - Сколько вакансий связано с анализом данных?
  - Насколько востребованы junior-специалисты?
  - Какие навыки требуются чаще всего?
  - В каких городах и компаниях сосредоточен рынок?
  - Сколько ключевых навыков в среднем указывают в DS-вакансиях?
- Сделать **осмысленные выводы и прогнозы**, а не только получить числовые ответы

---

## Используемые данные

В проекте используются следующие таблицы:

- **vacancies** — основная таблица вакансий  
  - название вакансии  
  - зарплатная вилка  
  - ключевые навыки  
  - опыт работы  
  - тип занятости  
  - регион  

- **areas** — справочник регионов  

- **employers** — справочник работодателей  

- **industries** — справочник сфер деятельности  

- **employers_industries** — связь работодателей и сфер деятельности (many-to-many)

## Ключевые этапы анализа

### 1. Общий обзор рынка вакансий
- Общее количество вакансий и работодателей
- Распределение по сферам деятельности

### 2. Выделение вакансий Data Science
DS-вакансии определялись по наличию в названии ключевых слов:
- `data scientist`
- `data science`
- `исследователь данных`
- `ML` (исключая HTML)
- `machine learning`
- `машинн%обучен%`

### 3. Анализ junior-вакансий
Junior-позиции определялись по условиям:
- наличие слова `junior` в названии
- опыт «Нет опыта»
- тип занятости «Стажировка»

### 4. Анализ ключевых навыков
- Частота упоминаний Python и SQL/Postgres
- Подсчёт среднего количества навыков  

### 5. География рынка
- Распределение вакансий по городам
- Анализ централизации рынка

---

## Основные выводы

- Data Science — **нишевый, но премиальный сегмент** рынка (~3% вакансий)
- Пик спроса приходится на специалистов с опытом **1–3 года**
- Junior-позиций крайне мало из-за высокой стоимости обучения и рисков
- Python и SQL — обязательные навыки почти для всех DS-вакансий
- В среднем в DS-вакансии указывают **6–6.5 ключевых навыков**
- Более **70% вакансий сосредоточены в Москве и Санкт-Петербурге**
- Данные частично неполные (NULL зарплаты, сферы деятельности)

## Итог

Проект демонстрирует полный цикл аналитической работы:
от постановки вопросов и написания SQL-запросов  
до интерпретации данных, поиска причин наблюдаемых тенденций и формулирования прогнозов.

Анализ показывает, что рынок Data Science в России:
**концентрированный, конкурентный и ориентированный на опытных специалистов**,  
с устойчивым потенциалом роста в ближайшие годы.
